# Quick Installation Guide

## How to Install Image Downloader Pro Chrome Extension

### Step-by-Step Installation (5 minutes)

1. **Extract the ZIP file**
   - Download `image-downloader-extension.zip`
   - Extract it to a folder on your computer
   - Remember where you extracted it

2. **Open Chrome Extensions**
   - Open Google Chrome browser
   - Type in address bar: `chrome://extensions/`
   - Press Enter

3. **Enable Developer Mode**
   - Look for "Developer mode" toggle in the top-right corner
   - Click to turn it ON (should be blue)

4. **Load the Extension**
   - Click "Load unpacked" button (top-left area)
   - Browse to the extracted folder `image-downloader-extension`
   - Select the folder and click "Select Folder"

5. **Verify Installation**
   - You should see "Image Downloader Pro" in your extensions list
   - Make sure it's enabled (toggle should be blue)

### Testing the Extension

1. Go to any website with images (e.g., https://unsplash.com)
2. Click the Image Downloader Pro icon in your toolbar
3. Click "Scan Page" button
4. You should see all images detected!
5. Click "Download Selected" to save them

### Troubleshooting

**Problem: "Load unpacked" button is greyed out**
- Solution: Make sure Developer Mode is enabled (toggle in top-right)

**Problem: Extension not appearing**
- Solution: Try refreshing the extensions page (F5)
- Make sure you selected the correct folder

**Problem: Can't find the extension icon**
- Solution: Click the puzzle piece ðŸ§© icon in toolbar
- Find "Image Downloader Pro" and pin it

**Problem: Permission errors**
- Solution: The extension needs permissions to access pages
- Click "Allow" when Chrome asks for permissions
